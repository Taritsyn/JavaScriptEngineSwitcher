   DESCRIPTION

   JavaScriptEngineSwitcher.Jint contains a `JintJsEngine` adapter (wrapper for the
   Jint (http://github.com/sebastienros/jint) version 4.6.0).

   =============
   RELEASE NOTES
   =============
   1. Jint was updated to version 4.6.0;
   2. Added support for .NET 10.

   DESCRIPTION
   ===========

   YantraJS (https://github.com/yantrajs/yantra) version 1.2.306).

   =============
   RELEASE NOTES
   =============
   1. YantraJS was updated to version 1.2.306;
   2. Added support for .NET 8.
